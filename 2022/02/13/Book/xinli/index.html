邓亚萍《心力》读书笔记

发表于2022-02-13|更新于2022-02-13|Book

心态提升

01.思维阻断：给情绪安装开关，就能控制焦虑

思维阻断 Thought Stopping

提炼出处
运动心理学家 Zinsser,N.,Bunker,L.。

简述
大脑对“禁止”的反应速度慢于“命令”，要自己“别想了”只会想得更多。正确的做法是提高自己“走神的能力”，在重要关头，靠转移注意力摆脱坏念头。

适用场景
考场中，因心态紧张而大脑空白，这个方法有助于及时止损，避免崩盘。

使用方法
设计你的“专属小动作”，在掉链子的时候，可以通过做这个动作适度走神。

瓦伦达效应 Wallenda Effect

简述
指为了达到一种目的而总是患得患失的心态

来历
卡尔·瓦伦达是美国著名的高空走钢索表演者，他在一次重大的表演中不幸失足身亡。
他的妻子事后说：我知道这次一定会出事。因为他上场前总不停地说：“这次太重要了，不能失败”。而以前每次成功的表演，他总想着走钢丝这件事本身，而不去管这个事可能带来的一切。

邓亚萍总结

第一，面对失误，最忌讳的就是反复提醒自己冷静，同时注意力却深陷其中。
第二，真正的赢家，会用思维阻断的方法，找到一件可以短暂分心并且全心投入的事情，主动给自己创造走神时刻。

02.失败清单：认清“失控感”，最大限度控制意外

失败清单 Failure List

提炼出处
心理学家 Kelly McGonigal.

简述
意外状况不可怕，“失控感”才可怕。所以，面对意外状况，要设计一个“失败清单”，这样一来，发生意外状况，大脑会认为“一切还在掌控中”。

适用场景
考题超出准备，遇到犀利的面试官，演讲、汇报时发生突发状况等。

使用方法
用游戏的心态，预设会发生的意外状况，并写在小卡片上。

邓亚萍总结

第一，面对意外状况，最忌讳“糟了！完蛋了！”这种心理暗示。
第二，赢家会用失败清单，说服自己确认“一切还在掌控中”。

03.迷信习惯：给自己设计一个小动作，做好心理暗示

运动员的小迷信 Superstitions

提炼出处
顶尖运动员的经验。

简述
运动场上充满太多不确定性，运动员为了增强“控制感”，都会用一些“幸运物” “幸运仪式” 来说服自己，相信自己是幸运的。和算命、锦鲤不同，这种“迷信”，是为了让自己有更好的表现。

适用场景
大考、面试、重要工作汇报前的最后时刻，让你不被外力干扰，对自己的表现更有自信。

邓亚萍总结

第一，迷信的重点不是你信不信，而是能够有效提升在不确定环境下，你对自己的控制力。迷信简单、粗暴、非理性，但是偏偏能管用。
第二，与其去盲从，听信一些别人给你推荐的小迷信，不如给自己精心设计一个小动作、小习惯。

04.自我激励：学会30度原则，才能时刻保持耐心

练习的心态 Practicing Mind

提炼出处
美国心理学者 Thomas M.Sterner.

简述
随着年龄的增长，我们只盯着目标和成绩，内心充满压力和焦虑。在学习一项新技能进，经常刚开始就放弃；或者面对生活或工作中的挑战，无法耐心又专心地应对。人的大脑对结果的暗示效率极低，控制力极弱。

适用场景
职场重要汇报、学校重要考试前，快速调整心态，增强自我鼓励。

使用方法
自我鼓励的内容应关注过程而不是结果。更具体来说，“打好这一分球”比胜利更重要。

邓亚萍总结

第一，自我激励不要想一步到位。你的大脑不会180度转向，相信你一定能赢。
第二，正确的方法是关注过程，激励自己把一件件小事做好，一次转30度。

05.角色定位：如何成为更强大的自己

角色定位 Role Definition

提炼出处
定位理论由美国著名营销专家艾·里斯（Al Ries）与杰克·特劳特（Jack Trout）提出

简述
人的自信，有时来自于“我是谁”的设定。你要帮自己找到一个好的角色，用这个角色的心态去面对挑战。

适用场景
需要达成一个目标，例如考试、考级、获得晋升，但缺乏自信或动力。

使用方法
给自己设定一个“角色定位”，并不断向期靠拢。

邓亚萍总结

人的自信，有时来自“我是谁”的设定。你要帮自己找到一个好的角色，用这个角色的心态去面对挑战。

06.如何用特定动作唤醒“强大人格”？

人设加强 Role Enhancement

提炼出处
“认知行为矫正法”，由心理治疗专家梅钦鲍姆（Donald Meichenbaum）提出，是用来改善负面负面情绪的心理治疗理论。

简述
不是自我认知指导行为，而是行为反过来塑造自我认知。刻意做一件和某人设相符的事，就会加强自我设定，让自己更加相信自己这样的人。

适用场景
考试、面试、谈判、汇报等，需要自信的场合。

使用方法
1.找到一个让自己有自信的人设；
2.刻意做和这个人设相符合的事，让你真实地相信自己。

邓亚萍总结

第一，很多时候，不是自我认知指导行为，是行为塑造自我认知。
第二，想要唤醒自己 “强大的一面”，要靠特定的动作。
第三，找到自己最强大的物质，主动做一些和这种物质相符合的事。

07.好的对手会让你遇强则强

竞争与潜能 Competition and Potential

简述
良性竞争会带来动力增长，激发潜能。
竞争会激发潜能，但是竞争和潜能不是线性关系，而是“倒U曲线”关系，要选择合适的对手。
你看待竞争的角度决定了你面对竞争的心态，竞争是挑战而不是威胁。
竞争的意义不是争，而是竞。不是为了打败对手，而是为了更早达到目标。

邓亚萍总结

首先，如果你遇到了一个竞争对手，那恭喜你，这可能标志着你在进步。
其次，怎么看待竞争取决于你的关注点，竞争是挑战、是机会、而不是威胁。
最后，选择一个合适的对手才能给你动力，也不让自己太有挫败感。
其实竞争永远不是最终目的，真正目的是不断挑战，自我成长。

效率提升

08.执行意图：大脑服从“自动程序”，对抗拖延症

执行意图 Implementation Intention

提炼出处
纽约大学心理系教授 Peter Gollwitzier

简述
拖延症产生的一大原因，是大脑会在决策时纳入各种意外因素进行考量。也就是说，偷懒总会战胜坚持。因此，如果能有针对具体场景的“预先设定”，大脑就可以路过决策阶段，自动自发去完成你设定的任务。

适用场景
锻炼、学习场景、处理每日待办琐事。

使用方法
在大脑里设定公式：“每到X场景，就要做Y事”。

邓亚萍总结
第一，大脑不喜欢僵局，大脑喜欢你给它具体的指示。
第二，要给大脑明确的指示：我到X场景，就必须做Y事。

09.例外控制：如何即保证休息，又保证高效

例外控制 Exception Management

提炼出处
纽约大学心理系教授 Gabriele Oettingen.

简述
人天生具有惰性。因此，在制定计划之前，要预先设想执行过程中可能遇到的阻碍，并为这些阻碍设置好备案。
比如计划跑步，就必须预料到自己可能会加班太累、不愿出门，备案可以是在家仰卧起坐。

适用场景
读书太疲倦，需要打个盹；工作压力太大，想刷会儿抖音。恰当地应对疲倦、压力，能够张弛有度。

使用方法
预先设想可能发生的例外状况，并提前做好备案；如果发生X例外情况，那么就应该做Y事。

邓亚萍总结

第一，不要强迫自己一直工作或学习，主动放松才能更快进步。
第二，休息的时候，感性大脑很容易就会接管大脑，让人控制不了惰性。
第三，可以设定一个“如果发生X例外情况，那么就应该做Y事”的条件句，拒绝感性接管大脑。
此外，WOOP(wish,outcome,obstacle,plan)思考法是对习惯养成和计划执行非常有帮助的一种方法。
